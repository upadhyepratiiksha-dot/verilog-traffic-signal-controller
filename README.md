# Traffic Light Controller

## Overview

This project implements a Traffic Light Controller using Verilog HDL based on a Finite State Machine (FSM). The controller manages traffic signals by transitioning through predefined states with appropriate timing. The design is functionally verified using a dedicated Verilog testbench.

## Features

- FSM-based traffic signal control
- Sequential state transitions
- Synchronous design
- Reset functionality
- Modular RTL implementation
- Functional verification using a Verilog testbench
